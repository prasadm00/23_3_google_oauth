

## How to run 
- 1. create `.env` file and inclue
 `CLIENT_APP_ID`,
  `CLIENT_APP_SECRET`,
  `REDIRECT_URI=/auth/google/secrets`

  2. install the dependencies by
  `npm i`
  3. run the app by
  `nodemon server.js`
  4. visit `localhost:3000/auth`
